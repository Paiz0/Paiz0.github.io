Some of my projects

    Game: Software Engineering course group project-- Knight Horse

    In this project, I was tasked to form and make a group project for a course at CSUF called "Software Engineering CPSC362".

    In this project, me and five others teamed up to make a game that we worked on through-out the fall semester at CSUF of 2021.

    This game revolves around a player on horseback that travels through the world constantly being bombarded by enemies-- left, right, and center.

    This game is developed using Unreal Engine 4, as a group we had members that were both familiar and unfamiliar with the software which presented a learning curve to get all the members on the same page with equal work loads.

    This project helped me learn a lot about UE4 development and API development. We developed an API system that allowed users to call our system and retrieve our highscore leaderboards at anytime.

    This call would return a player's name and their respective scores which was used by another group to host a leaderboard of scores on their pizza website.
